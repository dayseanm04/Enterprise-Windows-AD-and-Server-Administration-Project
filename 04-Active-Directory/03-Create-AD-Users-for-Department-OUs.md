## Objectives

## Steps to Create Users

### 1. Open Active Directory Administrative Center
- Open **Server Manager**
- Click **Tools**
- Select **Active Directory Administrative Center**

### 2. Navigate to Users OU
- Expand `corp (local)`
- Select **Users-OU**
